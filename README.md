# RSA_algorithm_py
RSA cryptography implementation in python

how the encryption work
1. firstly we input the message the we want to encrypt in encrypt_ms.py
2. encrypt_ms.py will put that message through the RSA encryption algorithm
3. and then encrypt_ms.py will change the output from RSA algorithm to binaries
4. and then that binaries message will be stored in Rsa_ms.json

how the decryption work
1. firstly we input the id of the encrypted message that stored at the Rsa_ms.json
2. secondly decrypt_ms.py will take the binaries message from Rsa_ms.json
3. and then encrypt_ms.py will change that binaries into decimal values
4. and change that values into characters, and print the output
