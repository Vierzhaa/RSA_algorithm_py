# RSA_algorithm_py
RSA cryptography implementation in python

How the Encryption Works

1. First, we input the message that we want to encrypt in encrypt_ms.py.
2. encrypt_ms.py then processes the message using the RSA encryption algorithm.
3. After that, the encrypted  is converted into binary format.
4. The binary message is then stored in Rsa_ms.json.

How the Decryption Works

1. First, we input the ID of the encrypted message stored in Rsa_ms.json.
2. Next, decrypt_ms.py retrieves the binary message from Rsa_ms.json.
3. Then, decrypt_ms.py converts the binary data back into decimal values.
4. Finally, those decimal values are converted into characters, and the decrypted message is printed as output.
